# 编译和链接

## 预处理

    gcc -m32 -E kernel/main.c -I include/ > test.c

```c
# 0 "kernel/main.c"
# 0 "<built-in>"
# 0 "<command-line>"
# 1 "/usr/include/stdc-predef.h" 1 3 4
# 0 "<command-line>" 2
# 1 "kernel/main.c"
# 1 "include/onix/onix.h" 1

void kernel_init();
# 2 "kernel/main.c" 2

int magic = 20220205;
char message[] = "hello onix!!!";
char buf[1024];


void kernel_init()
{
    char *video = (char *) 0xb8000;
    for (int i = 0; i < sizeof(message); i++)
    {
        video[i * 2] = message[i];
    }

}
```

## 汇编

    as -32 test.s -o test.o

    readelf -a test.o

## 链接

**/usr/lib/gcc/x86_64-linux-gnu/11/collect2**

    ld -m elf_i386 -static test.o -o test.out -e kernel_init

    readelf a test.out
